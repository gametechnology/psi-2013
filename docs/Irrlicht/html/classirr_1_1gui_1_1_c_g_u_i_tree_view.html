virtual void irr::gui::CGUITreeView::setIconFont ( IGUIFont *  font)
virtual

Sets the font which should be used as icon font. This font is set to the Irrlicht engine built-in-font by default. Icons can be displayed in front of every list item. An icon is a string, displayed with the icon font. When using the build-in-font of the Irrlicht engine as icon font, the icon strings defined in GUIIcons.h can be used.

virtual void irr::gui::CGUITreeView::setImageList ( IGUIImageList *  imageList)

Sets the image list which should be used for the image and selected image of every node. The default is 0 (no images).
