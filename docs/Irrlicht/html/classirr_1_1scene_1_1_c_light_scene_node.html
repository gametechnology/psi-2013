Detailed Description

Scene node which is a dynamic light. You can switch the light on and off by making it visible or not, and let it be animated by ordinary scene node animators.

void irr::scene::CLightSceneNode::setRadius ( f32  radius)
virtual

Sets the light's radius of influence. 

Outside this radius the light won't lighten geometry and cast no

shadows. Setting the radius will also influence the attenuation, setting it to (0,1/radius,0). If you want to override this behavior, set the attenuation after the radius.
