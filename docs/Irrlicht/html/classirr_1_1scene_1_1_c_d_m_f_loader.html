loads dynamic lights present in this scene.

Note that loaded lights from DeleD must have the suffix dynamic_ and must be pointlight. Irrlicht correctly loads specular color, diffuse color , position and distance of object affected by light.

Returns
number of lights loaded or 0 if loading failed.

loads water plains present in this scene.

Note that loaded water plains from DeleD must have the suffix water_ and must be rectangle (with just 1 rectangular face). Irrlicht correctly loads position and rotation of water plain as well as texture layers.

Returns
number of water plains loaded or 0 if loading failed.
