\subsection{Detailed Description}
An interface for easy manipulation of meshes. 

\begin{DoxyVerb}Scale, set alpha value, flip surfaces, and so on. This exists for fixing
\end{DoxyVerb}
 problems with wrong imported or exported meshes quickly after loading. It is not intended for doing mesh modifications and/or animations during runtime.
